# AI-Paper-Research-Assistant

## ⚙ 실행 방법

### 1. 실행 환경 설정

#### 1.1 레포지토리 클론

```bash
$ git clone https://github.com/juyoung-song/AI-Paper-Research-Assistant.git
```

#### 1.2 가상 환경 생성

```bash
$ python -m venv .venv
$ source .venv/bin/activate # macOS
$ .\venv\Scripts\activate # windows
(.venv) $
```

#### 1.3. 의존성 설치

```bash
(.venv) $ pip install -r requirements.txt
```
